# Caution
Please, please please do NOT blindly trust anyone on the internet and run whatever script they give you<br>
Read what the script does before running it<br>

### About this script
I know very little about Bash scripts,<br>
This script draws heavily on the work of ChrisTitus's [linutil](https://github.com/ChrisTitusTech/linutil)<br>

### How to run this script
After reading what it does,<br>
Open up your terminal, type

	curl -fsSL https://github.com/ChesterTsai/NLIS/raw/main/main.sh | bash

or try out dev branch (not recommended)

	curl -fsSL https://github.com/ChesterTsai/NLIS/raw/dev/main.sh | bash

### Also check out Nightlight's offical website!
